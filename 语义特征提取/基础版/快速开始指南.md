## 步骤1：准备AST JSON文件

首先需要将Python源代码转换为AST JSON格式。你可以使用Python的`ast`模块：

```python
import ast
import json

def python_to_ast_json(source_file, output_file):
    """将Python源代码转换为AST JSON"""
    with open(source_file, 'r', encoding='utf-8') as f:
        source_code = f.read()
    
    # 解析AST
    tree = ast.parse(source_code)
    
    # 转换为JSON格式（需要自定义序列化）
    ast_dict = {
        "metadata": {
            "source_file": source_file,
            "version": "1.0"
        },
        "ast": ast_to_dict(tree)
    }
    
    with open(output_file, 'w', encoding='utf-8') as f:
        json.dump(ast_dict, f, indent=2, ensure_ascii=False)

def ast_to_dict(node):
    """递归转换AST节点为字典"""
    result = {
        "type": node.__class__.__name__,
        "id": id(node),
        "children": []
    }
    
    # 添加节点特定属性
    for field, value in ast.iter_fields(node):
        if isinstance(value, list):
            for item in value:
                if isinstance(item, ast.AST):
                    result["children"].append(ast_to_dict(item))
        elif isinstance(value, ast.AST):
            result["children"].append(ast_to_dict(value))
        else:
            result[field] = str(value) if value is not None else None
    
    return result

# 使用示例
python_to_ast_json('测试示例_容器逃逸代码.py', '测试示例.ast.json')
```

### 风险等级检查脚本

```python
# check_risk_level.py
import json
import sys

def check_risk(report_file):
    with open(report_file, 'r', encoding='utf-8') as f:
        report = json.load(f)
    
    risk_level = report['risk_level']
    
    if risk_level in ['CRITICAL', 'HIGH']:
        print(f"❌ 检测到{risk_level}级别的容器逃逸风险！")
        print(f"   {report['summary']}")
        sys.exit(1)  # 失败退出
    elif risk_level == 'MEDIUM':
        print(f"⚠️  检测到中等风险，建议人工审查")
        print(f"   {report['summary']}")
        sys.exit(0)  # 警告但通过
    else:
        print(f"✅ 未检测到明显的容器逃逸风险")
        sys.exit(0)

if __name__ == '__main__':
    check_risk(sys.argv[1])
```

## 常见问题

### Q1: 如何处理大量文件？

```bash
# 批量处理脚本
for file in samples/*.py; do
    base=$(basename "$file" .py)
    python convert_to_ast.py "$file" "ast_json/${base}.ast.json"
    python semantic_feature_extractor.py "ast_json/${base}.ast.json" -r "$file"
done
```

### Q2: 如何调整检测规则？

编辑 `semantic_feature_extractor.py` 中的以下常量：
- `CONTAINER_ESCAPE_MODULES`：添加新的敏感模块
- `CONTAINER_ESCAPE_PATTERNS`：添加新的正则表达式模式
- `ESCAPE_RISK_RULES`：调整风险等级映射

### Q3: 如何减少误报？

1. 使用白名单机制过滤已知安全的代码模式
2. 结合上下文分析（如检查函数参数）
3. 调整风险分数阈值
4. 人工审查中等风险的样本
