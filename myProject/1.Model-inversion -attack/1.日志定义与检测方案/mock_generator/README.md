这部分的工作是：
    1.模拟生成数据：恶意程序样例与对应执行日志样例。
    2.对原始日志数据的预处理：从stuout格式的原始日志数据中提取出关键检测点信息并输出结构化的、json格式的日志信息。
为后面的针对于结构化的日志检测方案打基础。

## 生成器脚本 (log_generator.py): 
批量、随机地生成多种模型逆向攻击的Python代码及其对应的stdout格式日志文件。这些模拟场景将符合海洋科学计算的特点，并且日志中会蕴含您定义的关键检测特征。


## 解析器脚本 (log_parser.py): 
读取生成器产生的原始stdout日志文件，提取出关键特征，并将其转换、输出为结构化的JSON格式。
