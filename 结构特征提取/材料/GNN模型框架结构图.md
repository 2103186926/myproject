## 详细模块说明

### CFG-GNN模块（函数级）
```
输入：CFG图 (N个节点，16维特征)
│
├─ 第1层：GCNConv(16→32) + ReLU + Dropout(0.5)
│   └─ 图卷积：H¹ = σ(D̃^(-1/2) Ã D̃^(-1/2) H⁰ W⁰)
│
├─ 第2层：GCNConv(32→32) + ReLU + Dropout(0.5)
│   └─ 图卷积：H² = σ(D̃^(-1/2) Ã D̃^(-1/2) H¹ W¹)
│
├─ 第3层：GCNConv(32→64)
│   └─ 图卷积：H³ = D̃^(-1/2) Ã D̃^(-1/2) H² W²
│
└─ 全局平均池化：h_cfg = (1/N)Σh_i³
   └─ 输出：64维函数级嵌入
```

### FCG-GNN模块（程序级）
```
输入：增强的FCG图 (M个节点，80维特征)
│
├─ 第1层：GATConv(80→32) + ReLU + Dropout(0.5)
│   ├─ 注意力系数：α_ij = softmax(LeakyReLU(a^T[Wh_i||Wh_j]))
│   └─ 特征聚合：h'_i = σ(Σ α_ij Wh_j)
│
├─ 第2层：GATConv(32→64)
│   ├─ 注意力系数：α_ij = softmax(LeakyReLU(a^T[Wh_i||Wh_j]))
│   └─ 特征聚合：h'_i = Σ α_ij Wh_j
│
├─ 全局池化：h_fcg = pool(h_1, h_2, ..., h_M)
│
└─ 分类器：Linear(64→2)
   └─ 输出：程序级嵌入 + 分类logits
```

### 特征融合流程
```
对于FCG中的每个函数节点 f_i：
│
├─ 步骤1：查找对应的CFG
│   └─ cfg_i = CFG_dict[f_i.name]
│
├─ 步骤2：生成CFG嵌入
│   ├─ 如果CFG存在：h_cfg = CFG-GNN(cfg_i)  [64维]
│   └─ 如果CFG不存在：h_cfg = zeros(64)     [零向量]
│
└─ 步骤3：特征拼接
    └─ x'_i = concat(x_i[16维], h_cfg[64维]) = [80维]
```

## 训练损失函数

### CFG-GNN训练（阶段一）
```
L_cfg = BCE(边预测, 真实边)

其中：
- 正样本：真实存在的边
- 负样本：随机采样的不存在的边
- BCE：二元交叉熵损失
```

### FCG-GNN训练（阶段二）
```
L_fcg = L_edge + λ·L_cls

其中：
- L_edge：边预测损失（BCE）
- L_cls：分类损失（交叉熵）
- λ：平衡系数（通常设为1.0）
```

## 模型配置参数

| 模型 | 架构 | 层数 | 输入维度 | 隐藏维度 | 输出维度 | Dropout | 池化方式 |
|------|------|------|----------|----------|----------|---------|----------|
| CFG-GNN | GCN | 3 | 16 | 32 | 64 | 0.5 | 全局平均 |
| FCG-GNN | GAT | 2 | 80 | 32 | 64 | 0.5 | 可配置 |

**训练参数**：
- 优化器：Adam
- 学习率：0.001
- 权重衰减：5e-4
- CFG训练轮数：50
- FCG训练轮数：50
- 批次大小：32

## 关键创新点

1. **层次化设计**：分别学习函数级和程序级特征
2. **特征融合**：CFG嵌入增强FCG节点特征
3. **注意力机制**：FCG-GNN使用GAT关注重要函数
4. **两阶段训练**：先训练CFG-GNN，再训练FCG-GNN
5. **信息传递**：从局部到全局的语义信息流动
