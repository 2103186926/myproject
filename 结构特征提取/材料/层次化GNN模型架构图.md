### 详细的模块说明图

**1. CFG-GNN模块详细架构**
```
输入: CFG图 (节点特征矩阵 X, 边索引 edge_index)
│
├─ 输入层
│  └─ GCNConv(in=16, out=32)
│     └─ 图卷积操作: X' = D^(-1/2) A D^(-1/2) X W
│     └─ ReLU激活
│     └─ Dropout(p=0.5)
│
├─ 中间层 (可配置层数)
│  └─ GCNConv(in=32, out=32)
│     └─ 图卷积操作
│     └─ ReLU激活
│     └─ Dropout(p=0.5)
│
├─ 输出层
│  └─ GCNConv(in=32, out=64)
│     └─ 图卷积操作
│
├─ 全局池化
│  └─ global_mean_pool(x, batch)
│     └─ 聚合所有节点特征
│
└─ 输出
   ├─ 节点嵌入 (N × 64)
   ├─ 图嵌入 (1 × 64)
   └─ 分类logits (1 × 2)
```

### 关键技术点标注

**CFG-GNN特点:**
- 使用GCN进行图卷积操作
- 3层网络结构 (输入层 + 中间层 + 输出层)
- 全局平均池化聚合节点信息
- 输出64维CFG结构特征

**FCG-GNN特点:**
- 使用GAT引入注意力机制
- 2层网络结构 (更深的网络捕获复杂调用关系)
- 可配置的全局池化策略
- 输出64维FCG结构特征

**特征融合特点:**
- CFG嵌入作为FCG节点的附加特征
- 零向量填充处理缺失的CFG
- 实现函数级到程序级的信息传递
