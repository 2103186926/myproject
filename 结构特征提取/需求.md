
# 背景：
针对Python容器逃逸代码的静态检测方法，该方法采用双通道并行特征提取架构，通过语义特征提取和结构特征提取两大模块，首先通过抽象语法树（AST）分析和模式匹配提取128维语义特征，涵盖敏感函数调用、攻击模式、路径配置和混淆检测四个维度；其次，通过构建控制流图（CFG）和函数调用图（FCG），利用层次化图神经网络（GNN）学习代码的结构特征，生成256维结构嵌入向量；最终将两类特征融合为384维综合特征向量。实现对容器逃逸代码的精准识别。
