# 测试用例生成器 V3 使用说明

## 概述

`test_case_generator_v3.py` 是海洋科学云计算平台测试用例生成器的最终版本，专门为 GNN 模型训练设计，能够批量生成大量多样化的测试数据。

## 主要特点

### 1. 大规模数据生成
- 默认生成 100 个样本（50 正常 + 50 恶意）
- 支持自定义数量，可生成数百甚至上千个样本
- 满足深度学习模型对数据量的要求

### 2. 多样化场景覆盖
涵盖 20+ 种海洋科学计算场景：
- CTD数据处理
- ADCP流速分析
- 卫星遥感数据处理
- 数值模式后处理
- 海洋声学数据分析
- 海洋生态模型
- 海洋化学分析
- 海浪数据处理
- 潮汐预报计算
- 海洋温盐环流分析
- 海底地形处理
- 海洋气象耦合
- 渔业资源评估
- 海洋污染扩散模拟
- 珊瑚礁监测分析
- 海冰数据处理
- 海洋微生物基因组分析
- 深海探测数据处理
- 海洋能源评估
- 海洋灾害预警

### 3. 复杂的函数调用关系
- 每个样本包含 5-15 个函数（随机）
- 80% 为类成员函数，20% 为独立函数
- main 函数调用所有定义的函数，形成完整的 FCG
- 每个函数内部包含复杂的控制流（循环、条件、异常处理），形成丰富的 CFG

### 4. 样本独立性和区分性
- 每个样本使用不同的场景和函数组合
- 随机化函数数量和调用顺序
- 确保样本之间有明显差异

### 5. 恶意代码多样性
涵盖 10 种容器逃逸攻击类型：
- file_system_escape（文件系统逃逸）
- privilege_escalation（权限提升）
- namespace_escape（命名空间逃逸）
- cgroup_manipulation（cgroup 操作）
- docker_socket_access（Docker socket 访问）
- proc_manipulation（/proc 操作）
- capability_abuse（能力滥用）
- mount_escape（挂载逃逸）
- device_access（设备访问）
- kernel_exploit（内核利用）

## 使用方法

### 基本用法

```bash
# 生成默认数量（50 正常 + 50 恶意 = 100 个）
python test_case_generator_v3.py

# 生成指定数量
python test_case_generator_v3.py --num_normal 100 --num_malicious 100

# 指定总数量（平均分配）
python test_case_generator_v3.py --total 200

# 指定输出目录
python test_case_generator_v3.py --output_dir ./my_test_cases --total 150

# 设置随机种子（保证可重复性）
python test_case_generator_v3.py --seed 123 --total 100
```

### 命令行参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `--output_dir` | str | `./test_cases` | 输出目录 |
| `--num_normal` | int | 50 | 正常代码数量 |
| `--num_malicious` | int | 50 | 恶意代码数量 |
| `--total` | int | None | 总数量（如果指定，将平均分配） |
| `--seed` | int | 42 | 随机种子 |

### 输出文件结构

```
test_cases/
├── normal/                          # 正常代码目录
│   ├── ocean_task_normal_001.py
│   ├── ocean_task_normal_002.py
│   └── ...
├── malicious/                       # 恶意代码目录
│   ├── ocean_task_malicious_001.py
│   ├── ocean_task_malicious_002.py
│   └── ...
├── labels.csv                       # 标签文件（用于分类训练）
└── statistics.json                  # 统计信息
```

### labels.csv 格式

```csv
file_path,label
结构特征提取/test_cases/normal/ocean_task_normal_001.py,0
结构特征提取/test_cases/normal/ocean_task_normal_002.py,0
结构特征提取/test_cases/malicious/ocean_task_malicious_001.py,1
结构特征提取/test_cases/malicious/ocean_task_malicious_002.py,1
```

- label=0: 正常代码
- label=1: 恶意代码

## 数据量建议

根据不同的训练需求，建议生成的数据量：

| 用途 | 建议数量 | 命令示例 |
|------|----------|----------|
| 快速测试 | 20-50 | `--total 50` |
| 小规模训练 | 100-200 | `--total 200` |
| 中等规模训练 | 300-500 | `--total 500` |
| 大规模训练 | 1000+ | `--total 1000` |

## 注意事项

1. **内存占用**：生成大量文件时注意磁盘空间
2. **随机种子**：使用相同的 seed 可以生成相同的数据集
3. **标签文件**：自动生成的 labels.csv 使用相对路径，确保与训练脚本在同一工作目录
4. **函数数量**：每个样本的函数数量在 5-15 之间随机，确保多样性

## 故障排除

### 问题：生成的文件太多，占用空间大
**解决方案**：减少生成数量或定期清理旧数据

### 问题：训练时 batch 中样本数量不足
**解决方案**：增加生成的样本数量，确保每个类别至少有 batch_size * 10 个样本

### 问题：想要生成特定场景的代码
**解决方案**：修改 `OceanScenarioTemplates.SCENARIOS` 列表，添加或删除场景
